## 3. Описание логической структуры  
### 3.1 Алгоритм программы

3.1.1 Первый игрок загадывает слово и рисует черту для каждой буквы в этом слове.

3.1.2	Второй игрок пытается отгадать слово по одной букве за раз.

3.1.3	Если второй игрок правильно отгадывает букву, первый игрок заменяет соответствующую черту этой правильной буквой. Если буква встречается в слове дважды, её нужно отгадать дважды.

3.1.4	Если второй игрок угадал неправильно, первый игрок рисует часть фигурки (начиная с головы).

3.1.5	 Если второй игрок отгадывает всё слово прежде, чем будет полностью нарисована фигурка он побеждает. Если нет, проигрывает.

3.1.6	За первого игрока выступает алгоритм программы, за второго игрока – пользователь.

### 3.2	Используемые методы

3.2.1	В программе использован метод ввода и вывода информации, посредством ввода угадываемой буквы слова через клавиатуру ПК и вывода информации о процессе игры через монитор ПК.

### 3.3	Структура программы

3.3.1	Программа представляет собой функцию hangman в которой хранится игра. Эта функция принимает в качестве параметра переменную word – это слово, которое должен отгадать второй игрок. В переменной wrong отслеживается сколько неправильных предложений сделал второй игрок.

3.3.2	Переменная stages представляет собой список со строками, которая используется, чтобы рисовать фигурку. Переменная remaining_letters отслеживает, какие буквы осталось отгадать.

3.3.3	Переменная letter_board используется для отслеживания подсказок, которые отображаются второму игроку.

3.3.4	Переменная win, предназначена для отслеживания, победил ли второй игрок. В начале игры эта переменная принимает значение – False.

3.3.5	Основной функционал игры реализован в цикле, который продолжается до тех пор, пока значение переменной wrong меньше, чем len(stages) – 1. Переменная wrong отслеживает количество неправильных букв, указанных вторым игроком. В случае, если количество неудачных догадок окажется больше, чем количество строк для изображения фигурки игра закончиться.


3.3.6	В случае победы функция выводит сообщение: «Вы выиграли! Было загадано слово…». Если игрок проиграл выводится фигурка и сообщение «Вы проиграли! Было загадано слово…»

## Приложение А. Код программы.

```python
import random

def hangman():
    word_list = ["лето", "совершенство", "корабли", "музыка", "кот"]
    random_number = random.randint(0, 4)
    word = word_list[random_number]
    wrong_guesses = 0
    stages = ["", "________      ", "|      |      ", "|      0      ", "|     /|\     ", "|     / \     ", "|"]
    remaining_letters = list(word)
    letter_board = ["__"] * len(word)
    win = False
    print('Добро пожаловать на казнь')
    while wrong_guesses < len(stages) - 1:
        print('\n')
        guess = input("Введите букву: ")
        if guess in remaining_letters:
            character_index = remaining_letters.index(guess)
            letter_board[character_index] = guess
            remaining_letters[character_index] = '$'
        else:
            wrong_guesses += 1
        print((' '.join(letter_board)))
        print('\n'.join(stages[0: wrong_guesses + 1]))
        if '__' not in letter_board:
            print('Вы выиграли! Было загадано слово:')
            print(' '.join(letter_board))
            win = True
            break
    if not win:
        print('\n'.join(stages[0: wrong_guesses]))
        print('Вы проиграли! Было загадано слово: {}'.format(word))

hangman()
```

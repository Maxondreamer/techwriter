# Руководство по созданию репозитория в системе управления версиями Git.

## 1. Создание локального репозитория
- откройте интерфейс командной строки Git bash.
- перейдите в домашнюю директорию текущего пользователя с помощью команды: `cd ~`
- создайте пустую директорию с помощью команды: `mkdir <название репозитория>`
- перейдите в созданную директорию с помощью команды: `cd <название репозитория>`
- проведите инициализацию нового пустого репозитория с помощью команды: `git init`

## 2. Создание нового файла с проектом
- создайте новый файл  в репозитории с помощью команды: `touch <название файла>`
- убедитесь, что новый файл создан в репозитории с помощью команды: `git status`
- зафиксируйте изменения в репозитории с помошью команды: `git commit -m "<комментарий>"`
- убедитесь в текущем состоянии репозитория с помощью команды: `git status`

## 3. Создание удалённого репозитория
- войдите в свой аккаунт на Github.
- перейдите в раздел **Repositories**.
- для создания удалённого репозитория нажмите кнопку **New**.
- введите название репозитория полностью совпадающее с названием локального репозитория.
- нажмите кнопку **Create repository**.
- убедитесь, что удалённый репозиторий создан.

## 4. Привязка удалённого репозитория к локальному
- подключите удалённый репозиторий к локальному с помощью команды: `git remote add origin git@github.com:<имя аккаунта>/<имя репозитория>.git`
- убедитесь, что удалённый и локальный репозиторий связаны с помощью команды: `git remote -v`
- отправьте файлы в удалённый репозиторий с помощью команды: `git push -u origin master`
- убедитесь, что ветка обновлена с помощью команды: `git status`

## 5. Добавление проекта в репозиторий
- отредактируйте созданный ранее файл.
- подготовьте необходимый файл к сохранению с помощью команды: `git add <название файла>`
- зафиксируйте изменения в репозитории с помошью команды: `git commit -m "<комментарий>"`
- убедитесь в текущем состоянии репозитория с помощью команды: `git status`
- передайте изменения из локального репозитория в удалённый с помощью команды: `git push`
